Colección Musical – Aplicación Modular

Aplicación web desarrollada para la asignatura de Arquitecturas y Modelos para el Desarrollo Web Ágil cuyo objetivo es gestionar una colección musical personal. El proyecto incluye un backend en Node.js + Express + MongoDB Atlas y un frontend en React (Vite).

Tecnologías utilizadas

Backend:
- Node.js
- Express
- Mongoose
- MongoDB Atlas
- Dotenv
- CORS

Frontend:
- React
- Vite
- Fetch API

Instalación y ejecución

1.Clonar el repositoroio:
git clone https://github.com/keinyhm/proyecto-coleccion-musical

2. Instalar dependencias:
Backend:
  cd backend
  npm install
Frontend:
  cd ../frontend
  npm install

3. Ejecución del proyecto
El backend, desde la carpeta backend:
npm run dev
se ejecuta en: http://localhost:3000

El frontend, desde la carpeta frontend:
npm run dev
se ejecuta en: http://localhost:5173
